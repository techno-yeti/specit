# End-to-end testing

**Scope:**

- Full user flows through the UI:
  - Visit landing page.
  - Register.
  - Login.
  - Access dashboard.
  - Logout.

**Tooling:**

- Use a browser automation framework (e.g. Playwright or similar).

**Requirements:**

- Run against a test environment with seeded data.
- Avoid hitting production systems.
- Clean up any created test users or data.
