# UI components specification

Define reusable patterns using Tailwind utility classes.

**Buttons:**

- Primary button:
  - Base classes: `inline-flex items-center justify-center rounded-md px-4 py-2 text-sm font-medium`
  - Colours: primary background, white text, hover/active states.
- Secondary button:
  - Neutral background, border, hover states.

**Alerts:**

- Variants: success, error, warning, info.
- Include icon area, message text, optional close button.

**Navigation bar:**

- Sticky top bar with brand on left, auth/user menu on right.
- Responsive: collapses into menu on small screens.

Document each component with example HTML snippets and class lists.
